# ES6 Classes

This project covers class syntax and object-oriented features introduced in
ECMAScript 6.

## Learning Objectives

- Define a class.
- Add methods to a class.
- Add and use static methods.
- Extend one class from another.
- Understand metaprogramming and symbols.

## Requirements

- Ubuntu 20.04 LTS
- Node.js 20.x.x
- npm 9.x.x or newer
- Files use the `.js` extension
- Code is tested with Jest
- Code is linted with ESLint

## Files

| File | Description |
| --- | --- |
| `0-classroom.js` | Defines a `ClassRoom` class with a maximum student size. |
| `1-make_classrooms.js` | Creates three `ClassRoom` instances. |
| `2-hbtn_course.js` | Defines a course class with getters, setters, and validation. |
| `3-currency.js` | Defines a currency class with a formatted display method. |
| `4-pricing.js` | Defines a pricing class with currency formatting and conversion. |
| `5-building.js` | Defines a base building class requiring subclass evacuation warnings. |
| `6-sky_high.js` | Defines a high-rise building subclass with an evacuation warning. |
| `7-airport.js` | Defines an airport class with a custom string tag. |
| `8-hbtn_class.js` | Defines primitive conversion behavior for a Holberton class. |
| `9-hoisting.js` | Fixes class ordering and exports a list of students. |
| `10-car.js` | Defines a car class that clones using species construction. |
