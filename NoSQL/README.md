# NoSQL

This project covers basic MongoDB shell commands and PyMongo helpers for the Holberton `NoSQL` tasks.

## Files

- `0-list_databases`: lists all MongoDB databases.
- `1-use_or_create_database`: switches to `my_db`.
- `2-insert`: inserts `Holberton school` into `school`.
- `3-all`: lists all documents in `school`.
- `4-match`: lists documents where `name` is `Holberton school`.
- `5-count`: prints the number of documents in `school`.
- `6-update`: adds the `address` field to matching documents.
- `7-delete`: deletes matching documents from `school`.
- `8-all.py`: returns all documents from a collection.
- `9-insert_school.py`: inserts a document from keyword arguments.
- `10-update_topics.py`: updates school topics by name.
- `11-schools_by_topic.py`: finds schools by topic.
- `12-log_stats.py`: prints MongoDB-backed Nginx log statistics.
